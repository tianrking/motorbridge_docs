# Mintlify Docs for Python Binding

This folder contains a dedicated Mintlify documentation site for `bindings/python`.

## Local preview

From repository root:

```bash
cd bindings/python/mintlify
npx mintlify dev
```

If you do not have Mintlify installed globally, `npx` is enough.

## Entry files

- Site config: `docs.json`
- Home page: `index.mdx`
- API docs: `api/*`
- Tutorials: `tutorials/*`
- Best practices: `best-practices/*`
- Reference: `reference/*`

## Scope

This site is focused on Python binding workflows and API usage patterns, aligned with current `motorbridge.core.Controller` and `motorbridge.core.Motor` interfaces.
